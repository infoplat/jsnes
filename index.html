游戏说明
  • 选择游戏... = 点击游戏列表下拉框选择
  • 暂停&继续 = 暂停或继续游戏
  • 重新开始 = 重新开始游戏
  • 打开&关闭声音 = 打开或关闭游戏声音
  • 放大&缩小 = 放大或缩小游戏窗口

游戏控制
  • A = K / Num-7
  • B = J / Num-9
  • 上 = W / Num-8
  • 下 = S / NUm-2
  • 左 = A / NUm-4
  • 右 = D / NUm-6
  • 开始 = ENTER / Num-1
  • 选择 = CTRL / Num-3
  • 控制说明 = 按键"/"左边为P1右边为P2
